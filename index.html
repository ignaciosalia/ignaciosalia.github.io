Surragate

Es una aplicación android en desarrollo, que simula un "escritorio remoto" de un disposotivo android a otro. Usando bluethood, esto permite realizar u operar algunas aplicaciones del dispositivo esclavo de manera remota. Está pensado para cuando se viaje que se pueda esconder un dispositivo y utilizar uno menos costoso. Así en caso de robo se puede proteger el dispositivo que se quiera, mientras el ladron se "va" con el señuelo.
